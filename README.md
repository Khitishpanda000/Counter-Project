# Counter-Project
A simple React-based counter application built using Vite. This project demonstrates the use of React hooks (useState) to manage state and handle user interactions such as incrementing and decrementing a counter value. It is a beginner-friendly project to understand the fundamentals of React state management and event handling.

# React Counter App

A simple and interactive counter application built using **React** and **Vite**. This project is perfect for beginners who want to understand how state management works in React using hooks.

Features
* 🔼 Increase counter value
* 🔽 Decrease counter value
* ⚡ Fast performance using Vite
* 🎯 Simple and clean UI
* 🧠 Demonstrates `useState` hook

## 🛠️ Tech Stack
* React
* Vite
* JavaScript (ES6+)
* CSS
  
## 💡 How It Works

* The app uses React's `useState` hook to manage the counter state.
* Clicking **Increase Count** increments the value.
* Clicking **Decrease Count** decrements the value.
* Each click updates the UI instantly.
